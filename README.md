# Projeto-Final-de-POO
Sistema de Gerenciamento de Hotel/Pousada desenvolvido em Java com arquitetura MVC, voltado para o controle de hóspedes, reservas, quartos, check-in e check-out. O projeto organiza as entidades e regras de negócio do sistema, separando as responsabilidades entre Model, View e Controller para facilitar a manutenção e a evolução da aplicação.
